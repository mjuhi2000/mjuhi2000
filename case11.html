1.C program code for Leap Year, this C program is used to check whether the given year of input is a leap year or not.

For Example-:

	 #include<stdio.h>
	 void main()
	 {
	   int year;
	   printf("enter the year");
	   scanf("%d",&year);
	   if(year%4==0)
	   {
	     printf("year is leap");
		 }
		 else
		 {
		   printf("year is not leap");
		   }
		}
